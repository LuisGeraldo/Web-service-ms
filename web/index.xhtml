Web servie para registrar materia

Acerca del proyecto:

Propósito:

El web service tiene como propósito almacenar materias universitarias en una base de datos.

Tecnología usada:

  • Web service ms es un proyecto desarrollado con java

  • Lenguaje de presentación: HTML, CSS

  • Lenguaje de programación: Java 8

  • Frameworks: JQuery, JSF

  • Servidor: GlassFish server

  • IDE: NetBeans

  • Controlador de versiones: Git

  • Base de datos: Java DB
